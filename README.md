# ReviewInsight AI

ReviewInsight AI is an AI-powered product review analysis application that helps businesses identify product strengths, weaknesses, and improvement priorities from customer reviews.

## Planned Features

- CSV review upload
- Aspect-based sentiment analysis
- Product variant comparison
- Evidence retrieval
- Improvement priority ranking
- AI-powered question answering

## Technology Stack

- Python
- FastAPI
- Pandas
- React
- TypeScript
- Tailwind CSS

## Current Status

Initial FastAPI backend setup completed.
